git bash 免密ssh 登录配置

2019-07-18
  1. 使用命令ssh-keygen ，生成私钥和公钥
  2. 私钥保存在本机~/.ssh下，公钥保存在服务器/root/.ssh下 密码直接回车，这样才能免密
  3. cat id_rsa.pub >> authorized_keys
  4. vim /etc/ssh/sshd_config 去掉#激活(AuthorizedKeysFile .ssh/authorized_keys)
  5. 重启ssh服务，systemctl restart sshd.service 或 service sshd restart
